⚡ SETUP CEPAT (5 MENIT):
═══════════════════════════════════════════════════════════════

1️⃣  BUAT GOOGLE SHEETS
    → Buka: sheets.google.com
    → Klik: + Blank
    → Nama: "PR Tracker Database"
    → Copy paste data dari TEMPLATE_GOOGLE_SHEETS.txt

2️⃣  PUBLISH SHEETS
    → File → Share → Publish to web
    → Pilih: Entire Document
    → Format: CSV
    → Centang: Automatically republish
    → Klik: Publish

3️⃣  COPY SHEET ID
    → Lihat URL: https://docs.google.com/.../d/SHEET_ID/edit
    → Copy bagian SHEET_ID
    → Contoh: 1abc123XYZ456

4️⃣  UPDATE HTML
    → Buka: pr-tracker-sheets.html (dengan Notepad)
    → Cari: const SHEET_ID = 'GANTI_DENGAN_SHEET_ID_ANDA';
    → Ganti dengan Sheet ID Anda
    → Save (Ctrl+S)

5️⃣  TEST!
    → Double-click: pr-tracker-sheets.html
    → Login sebagai Guru/Murid
    → Lihat data muncul!

✓ Muncul toast: "✅ Berhasil memuat X data dari Google Sheets"

❌ GAGAL JIKA:
═══════════════════════════════════════════════════════════════

✗ Muncul: "❌ Gagal memuat data dari Google Sheets"
  → Solusi: Pastikan sheet sudah di-publish

✗ Data tidak muncul
  → Solusi: Cek Sheet ID benar, sheet sudah di-publish

🔧 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════

Q: Data tidak update setelah edit di Sheets?
A: Tunggu 5-10 detik, lalu klik "🔄 Refresh"

Q: Sheet ID dimana?
A: Di URL Google Sheets, bagian setelah /d/ dan sebelum /edit

Q: Format tanggal salah?
A: Gunakan YYYY-MM-DD (contoh: 2026-04-15)

Q: Karakter Indonesia jadi aneh?
A: Pastikan Google Sheets di-save dengan encoding UTF-8
