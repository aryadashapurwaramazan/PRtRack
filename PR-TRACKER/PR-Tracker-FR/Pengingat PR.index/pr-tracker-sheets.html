⚙️ Setup Google Apps Script

📋 Langkah-langkah:
1. Buka Google Spreadsheet ini
2. Klik menu Extensions > Apps Script
3. Hapus kode yang ada, paste kode dari file GOOGLE_APPS_SCRIPT.js
4. Klik Deploy > New Deployment
5. Type: Web App, Execute as: Me, Access: Anyone
6. Klik Deploy, copy URL yang muncul
7. Paste URL di bawah ini lalu klik Simpan
URL ini disimpan di browser kamu (localStorage)
