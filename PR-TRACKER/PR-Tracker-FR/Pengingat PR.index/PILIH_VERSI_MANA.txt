Ada 2 versi yang tersedia:

═══════════════════════════════════════════════════════════════
  1️⃣  VERSI STANDALONE (pr-tracker.html)
═══════════════════════════════════════════════════════════════

✅ KELEBIHAN:
  • Tidak perlu internet
  • Tidak perlu setup Google Sheets
  • Langsung bisa digunakan
  • Import CSV dari Excel/Google Sheets
  • Data tersimpan di browser

❌ KEKURANGAN:
  • Data hilang jika clear browser
  • Tidak bisa kolaborasi real-time
  • Harus import manual setiap kali ada perubahan
  • Tidak ada backup otomatis

📌 COCOK UNTUK:
  • Guru yang bekerja sendiri
  • Tidak butuh kolaborasi
  • Jarang update data
  • Tidak ada koneksi internet stabil

🚀 CARA PAKAI:
  1. Double-click pr-tracker.html
  2. Login sebagai Guru
  3. Klik "📄 Download Template"
  4. Isi data di Excel
  5. Klik "📥 Import CSV"
  6. Selesai!


═══════════════════════════════════════════════════════════════
  2️⃣  VERSI GOOGLE SHEETS (pr-tracker-sheets.html) ⭐ RECOMMENDED
═══════════════════════════════════════════════════════════════

✅ KELEBIHAN:
  • Real-time sync dengan Google Sheets
  • Kolaborasi dengan banyak guru
  • Edit dari HP/laptop/tablet
  • Backup otomatis oleh Google
  • Auto-refresh setiap 30 detik
  • Data tidak hilang
  • Version history (bisa undo)

❌ KEKURANGAN:
  • Butuh internet
  • Perlu setup Google Sheets (5 menit)
  • Butuh akun Google

📌 COCOK UNTUK:
  • Banyak guru yang kolaborasi
  • Butuh akses dari mana saja
  • Update data sering
  • Butuh backup otomatis
  • Ada koneksi internet

🚀 CARA PAKAI:
  1. Buat Google Sheets
  2. Publish to web
  3. Copy Sheet ID
  4. Update pr-tracker-sheets.html
  5. Selesai!
  
  (Lihat: QUICK_START_GOOGLE_SHEETS.txt)
