🎯 UNTUK GURU:
═══════════════════════════════════════════════════════════════

CARA EDIT DATA:
1. Klik tombol "📊 Edit di Sheets" di web
2. Atau buka langsung Google Sheets Anda
3. Edit data (tambah/ubah/hapus baris)
4. Data otomatis sinkron ke web (tunggu 30 detik atau klik refresh)

FORMAT DATA DI GOOGLE SHEETS:
Kolom A: Nama Siswa
Kolom B: Kelas (X-A / X-B / X-C)
Kolom C: Judul PR
Kolom D: Mata Pelajaran (HTML / CSS / JavaScript / Lainnya)
Kolom E: Tanggal Diberikan (YYYY-MM-DD, contoh: 2026-04-15)
Kolom F: Deadline (YYYY-MM-DD, contoh: 2026-04-22)
Kolom G: Status (Sudah Dikumpulkan / Belum Dikumpulkan / Terlambat)
Kolom H: Nilai (0-100, boleh kosong)
Kolom I: Catatan (teks bebas, boleh kosong)

⚙️ PENGATURAN:
═══════════════════════════════════════════════════════════════

Auto-refresh: Setiap 30 detik
(Data otomatis update tanpa perlu klik refresh manual)

Jika ingin mengubah interval:
1. Buka pr-tracker-sheets.html dengan Notepad
2. Cari: const AUTO_REFRESH_INTERVAL = 30000;
3. Ubah angka (dalam milidetik):
   - 30000 = 30 detik
   - 60000 = 60 detik (1 menit)
   - 120000 = 2 menit
4. Save file


🔧 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════

❌ Data tidak muncul?
   → Cek koneksi internet
   → Pastikan Google Sheets sudah di-publish
   → Klik tombol "🔄 Refresh"

❌ Data tidak update setelah edit di Sheets?
   → Tunggu 5-10 detik (Google butuh waktu untuk publish)
   → Klik tombol "🔄 Refresh"
   → Cek apakah edit sudah di-save di Google Sheets

❌ Muncul error "Gagal memuat data"?
   → Cek koneksi internet
   → Refresh halaman (F5)
   → Coba lagi beberapa saat
